Design preview assets only.

These files are workbench output from skin design, role/medal generation,
and UI iteration. They are not runtime assets and are not referenced by
`mobile/miniprogram/skins.js`.

Directory layout:

- `skins/celestial/`: celestial skin drafts, previews, and screen captures
- `skins/dark-dungeon/`: dark-dungeon skin drafts and previews
- `skins/references/`: external/reference images used during skin exploration
- `workbench/shared-ui/`: shared UI preview images and component checks
- `workbench/roles/`: role generation output and role atlas previews
- `workbench/medals/`: medal atlas generation output

Runtime assets remain under:

- `mobile/assets/`
- `mobile/miniprogram/assets/`

When adding new generated images, place them here instead of the repo root.
